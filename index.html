一个人的成功是如何被运气决定的

发表于 2022-09-17 更新于 2022-09-20 本文字数： 1.6k 阅读时长 ≈ 4 分钟

引言

       这一期我们来看一项有趣的研究，它在数学上证明了成功是运气的结果，回答了成功究竟凭借自己的能力还是运气这一讨论不断的难题，而且是给出了这么一个不寻常的答案，也因此获得了今年的搞笑诺贝尔经济学奖。

        

         一般而言，人们对于能否成功的认知取决于两个因素：天赋和运气。前者代表了个人与生具来的各种能力，后者代表了外部的种种不确定性因素；从个人的经验来讲，很难说谁更重要，我们可以看到各种各样的观点，但是社会整体显然是认为前者、也就是一个人的天赋更为重要。

        为什么这么说呢？

        因为我们在社会的各个领域都可以看到这一现象：

        比如在教育领域，成绩优秀的学生会进入更好的学校、获得更好的教育资源；在工作领域，表现更好的员工更容易获得晋升；成功的经营者往往会得到资本的青睐；在学术领域，出过成果的研究者会得到更多的研究机会与赞助，等等……

        这些都是假设成功由天赋决定，成功者更有能力，所以将资源给成功者相比给所谓的“平庸者”能创造更大的价值。

        “运气是什么？成功者的每一步都是必然的！”

        然而，科学史上许多的重要发现都得益于运气——我们熟悉的就有青霉素、硫化橡胶、石墨烯……灵感的出现也只是神经网络的随机结果；

        从更大的整体层面来看，一些研究也表明了运气的重要性，最有代表性的就要数姓名的首字母排序。基于首字母排序是日常中常用的方法，因为它不会涉及到歧视，这就使得排在前列的人更容易获得关注和机会。

        外部运气的重要性在统计学上表露无疑。

        人类的智力或者说更广泛的天赋是有差别的，整体上呈现正态分布，也就是说能力极强或者极弱的人都只占了一小部分，而社会财富的分布却遵循了帕雷托法则，呈现为幂律分布，也就是穷人占了大多数，而极少数的富翁占据了主要的财富，天赋与财富不相匹配。

        在这一财富分配过程中难道不是运气在起作用吗？

模型

        为了证明这一点，研究者制作了一个简单的模型来模拟人生财富积累的路径。

        这个模型将代表个人的1000个小人放置在规定的区域内，固定好不再移动，并且按照自然的正态分布为他们赋予不同的天赋值和相同的初始资本；在上面添加了500个圆点，使它们随机在区域内移动，分为绿色与红色来代表幸运和不幸的事件，两者数量相同。

        小人遭遇到幸运或者倒霉事件后，会增加或者减少他们的初始资本，增减的幅度与天赋有关。

        研究者用圆点的移动来代表人生的20岁到60岁这一年龄段，每移动一次就相当于个人的6个月，总计移动80次，结束后统计他们的最终资本。

        结果是这样的。

        数据遵循了帕雷托法则，我们可以看到，大部分的人都挤在了图表的左侧，有近一半的小人，最终资本都在10以下，也就是最初的资本数之下。

        这很类似于现实的财富分配现状。

        那么这些最终资本在几百上千的小人，他们的天赋是最好的那一小部分人吗？

        答案是否定的。

        这个图表统计了天赋和最终资本间的函数关系，很显然，拥有最多资本的人，他们的天赋都分布在平均值附近，最成功者的天赋只是略高于平均水平，而那些最有天赋的人反而远远不如。

        可见，天赋的大小对于财富的积累并没有很强的解释力，尽管天赋的大小影响了个人利用幸运事件的效率，但这一影响对运气的影响来说毫不明显。

        同时的图表也显示了随机事件和最终资本间的函数关系，显然，资本量与随机事件之间有着强烈的相关性，最成功的人也是最幸运的人，反之，最不幸的人也是最失败的人。

结论

         这看起来是一个很令人沮丧的结论，幸运的人更加成功，个人能力的影响被淹没了，而社会又在强化这一过程，给予幸运的人更多的机会和资源，形成“马太效应”。     

        只不过，从微观个人的角度来看就是另一回事了，因为天赋越强越能利用好机会，中等天赋的成功者只是因为基数更大；所以从概率上来讲，一个更有天赋的人更可能获得成功，对于个人而言，  出身、机会等运气事件是不可控的，但是提高自己的能力、积极与他人接触，以此来碰到更多的机会、更好地利用机会却是可以控制的。

                

为什么要用地理、动物来命名流行病？

发表于 2022-08-09 更新于 2022-08-11 本文字数： 1.9k 阅读时长 ≈ 5 分钟

一、引言

  最近 ，全球权威医学杂志《新英格兰医学》发表的一篇文章指出，在中国山东、河南发现了一种新型病毒，而且已经出现了动物传人的现象，它被作者命名为“瑯琊病毒/Langya Virus（LayV）”。

  但是，在病毒的威胁力如何还没有明朗时，它的名字就已经遭到了不少人的吐槽和反对。瑯琊作为山东地区的古地名，用作某个病毒的命名，被认为是一种污名化，尤其是在经过新冠肺炎以“武汉”命名的争论之后，人们对于这样的命名更为敏感。

  某种流行疾病以地名作为名称是一个传统，上世纪以来的几次大流行，比如1918年的西班牙流感、1957年的亚洲流感、1968年的香港流感，2009年的甲型H1N1流感——也被称为墨西哥流感或者北美流感，都是以地名进行命名的，其它的还有中东呼吸综合症（MERS）、埃博拉病毒等等也是如此。

  除了地名以外，动物名也容易中招，比如今年爆发的猴痘病毒，之前爆发过的禽流感、猪流感，还有马脑炎等等，很明显地是以某种动物来命名的。

二、命名带来的麻烦

  不过，这一命名的传统却带来了很多麻烦。

  2009年爆发的甲型H1N1流感最初就被叫作“猪流感”，因为它是起源于猪，是普通猪流感与禽流感、人流感的混杂结果。尽管它的传播主要是靠人与人之间的传染，这个名字也使人们谈猪色变，沉重地打击了猪肉及相关的行业：一些国家开始禁止进口猪肉，埃及出现了大量屠杀猪群的现象，股票市场上的猪肉期货大幅下降，普通市民在消费时也避免购买猪肉产品。

  这样恶劣的影响引起了猪肉行业和世界动物卫生组织的抗议，迫使世界卫生组织把名字改为“甲型H1N1流感”。与此同时，因为这一流感最初是在墨西哥发现的，“墨西哥流感”的称呼自流感爆发以来，就一直在流传，这样的名字受到了墨西哥政府的反对，而且也造成了严重的社会影响。

  在墨西哥的邻国——美国，一直以来对于墨西哥劳工和非法移民都有着很深的误解和恐惧，这样的名字更是加重了这一偏见：一些人将墨西哥移民视为病毒传播的罪魁祸首，呼吁限制边境通行，更有甚者借此来攻击墨西哥人，以为流感在墨西哥的大范围传播是因为他们的遗传基因更加“脆弱”。

  这一现象也在新冠的流行中得到重演，亚裔被污名化、被贴标签，与之相关的种族仇恨犯罪大幅上升。

  世卫组织在这一方面也承受了相当的压力，为此，世卫组织在2015年，提出了新的人类传染病命名方法，希望减少疾病的命名对于旅游、贸易、动物福利带来的不利影响，避免对任何的地区或者群体造成冒犯。

  世卫组织建议疾病的名称最好不要涉及地理区域、人名、动物或食物、某类人口或者职业、耸人听闻的词语，比如中东呼吸综合症、埃博拉、禽流感等名字都是不合适的。只不过，这样的建议并没有得到广泛的认同和执行。

三、为什么这样命名

  之所以会出现这样的负面影响，是因为不同群体对于同一名称的理解是不一样的。对于一般的公众来说，传染病的名字表明了它的传播情况，猪流感是就由猪进行传播的，墨西哥流感就表明了要远离墨西哥人；但是对于流行病学家来说，用地理名词或者动物种类来命名一种流行疾病，是为了追溯它的源头，“猪流感”是因为这一病毒具有普通猪流感的遗传片段，不表明它是由猪传播的。

  所以，对于研究者来说，这样的命名是很合理的，而且有利于区分相似的疾病，比如埃博拉和马尔堡病很相近，如果不以地名命名的话，可能会变成丝状病毒出血热1型和丝状病毒出血热2型。

  而且，要创造一个完善不冒犯他人的名字是很困难的，不太可能考虑到所有情况。2002年的非典，也被称作“严重急性呼吸道症/Severe Acute Respiratory Syndrome”，缩写为“SARS”，这是一个极中立的命名，但是它的缩写却与香港“特别行政区/Special Administrative Region”的缩写“SAR”只有一个字母之差，因而引起了一些不满。同样，以发现地命名的”诺沃克病毒/Norwalk”被改为”诺如病毒/Noro”，但这又引起了日本人的抗议，因为在日本这是一个常见的姓氏。

  尽管有诸多不便，世卫组织提出的规则确实是更好的一种方向，不合适的命名会伤害到一些社会群体的情感，而且可能在公共传播中产生恶性的循环，造成难以估量的损失。

眼睛图案促进亲社会行为

发表于 2022-08-04 更新于 2022-09-25 本文字数： 1.8k 阅读时长 ≈ 5 分钟

一、引言

  动物为了生存，进化出了许多策略来躲避捕食者，比如伪装色、警告色、威吓等，其中一个例子是蝴蝶翅膀上的眼状斑纹，这是一种有效提高生存率的手段，研究者为了解释背后的机制，提出了多个假设：

  • 吸引攻击转移到非重要的身体区域；

  • 模仿捕食者敌人的眼睛来恐吓对方；

  • 鲜明、独特的特征会使捕食者回避；

  • 欺骗捕食者认为自己被发现。

  尽管我们没有看到哺乳动物使用这种策略，但实验证明了它对哺乳动物也会起作用，人类也不例外。当然了，后者不是在说作为捕食者的人类，而是指作为社会中一员的文明人，有许多实验表明，只是利用画出来的眼睛图案就可以让人变得更乐善好施。

二、实验

  这个结论是通过一个精心设计的游戏得出来的，名字叫做“独裁者”。

  规则很简单：参与游戏的玩家会被随机两两分配，其中一个玩家会得到10美元，并需要在20秒内决定将多大的一部分捐给另一位配对的玩家，剩下的就归自己所有；玩家只能进行一轮游戏，全程匿名，结束后悄悄地拿钱走人。

  为了防止玩家间的互动，玩家会被随机地安排在一个计算机实验室里，相互间有空位、桌面隔板，禁止相互间的眼神或语言交流，整个过程在计算机上进行——其中一部分显示有眼睛图案，另一部分作为对照。

  最后的结果是，看到眼睛的参与者中大约有80%都捐出了钱，与之相对的是，其它参与者中只有大约一半捐出了自己的钱，就数额来说，前者也比后者多，平均每个人给出了近4美元，比后者多出55%。显然，假的眼睛图案会让人变得更慷慨。

  尽管这是一个通过实验室中的游戏得出来的结论，但是它完全可以应用在社会的各个场景之中。

  后来的研究者把实验搬出了实验室，观察它在自然的慈善活动中的表现。他们在一个大型超市进了为期十一周的实验，内容是在超市的每个收银台上都放一个募捐桶，一半画上眼睛的图案，一半画上星星的图案，观察这些桶收集到的善款多少。

  与之前的游戏一样，画有眼睛的募捐桶中的捐款比画有星星的桶多了48%，而且这一效果在超市冷清的时候更加明显，顾客少的时候前者比后者会多出59%，似乎其他人的存在会稀释眼睛的效果。

三、抑制反社会行为

  实际上，这一个理论在另一方面的应用更加受人关注，也更加有效——也就是对于不道德或者说反社会行为的抑制，比如让人讨厌的乱丢垃圾、自行车盗窃等行为，它在这方面受到更大的关注是有原因的，因为一些政府组织和社会团体已经在利用这一效应，而且得到了不错的反馈。

  英国一些地区的警察声称他们利用画有眼睛的海报减少了犯罪，环保组织在各地的高速公路服务站上张贴的海报使乱扔垃圾的现象减少了23%，这也被英国政府所认可，并在国家的减少垃圾策略中得到推荐。

四、原因

  我们确实地可以看到简单的眼睛图案就可以促进道德行为，抑制不道德行为，但是它是怎么做到的？为什么会这样？

  答案很简单，那就是它让周围的人误以为自己被注视，从而调整自己的行为，就像动物欺骗捕食者以为自己被发现了一样。我们的大脑中有专门负责这一部分的神经元，当看到人脸、眼睛或者类似的图案时，它们会不自主地激活，从而调整自己的行为，这可能是为了在社会中保护自己的声誉以更好地生存下来。

  这就是为什么看到这个图案的人会阻止自己的不道德行为，因为他们潜意识里认为在这里扔垃圾或者偷东西会损害自己的声誉，是不利于自己的。

五、一些反对意见

  这也解释了为什么这一效应在防止不道德行为方面，会比促进良好行为方面更为有效。一开始我们提到眼睛的图案使人变得更慷慨，但它不是一直有效的，而且它也不是提高了捐款数额，只是提高了捐款的可能性，因为眼睛的图案不是让人变得更好，而是让人不自主地考虑社会的影响，你过于突出不会让人更喜欢你，反而可能会有反效果。

  这一效应还面临着其它敌人，那就是习惯化，“当一个刺激反复或长时间出现时，对该刺激的反应性下降”，周围的人过多时也会减弱对这一刺激的反应。

  当然了，这还不能构成我们否定它的原因，因为哪怕只有微弱的效应，在较大的基数和时间跨度下，它也可以造成显著的结果，更何况它只需要你画一双眼睛。

网络引战

发表于 2022-07-29 更新于 2022-07-30 本文字数： 1.8k 阅读时长 ≈ 5 分钟

一、引言

  键盘侠、杠精、喷子、水军，如果你是社交网站的活跃用户，那你就不可能没有听说这些角色，也很难不会遇到他们。在明星偶像、在社会问题、在突发事件、甚至于在学术讨论之下，都可以看到他们的身影，他们常常通过挑衅、讽刺、辱骂的手段来引起争端，形成骂战，使讨论区变得乌烟瘴气。

  这种行为就是引战，在英文中这些人被比作“Troll”，一种传说的巨魔，由此可见他们是多么地令人讨厌。然而，在现实中，这些人并不比其他人更令人憎恶，因为他们不会轻易地当面去嘲讽别人，但是网络改变了他们的行为。

二、网络去抑制

  为什么同一个人在线上、线下会表现得截然不同？

  心理学家约翰·舒勒提出的“网络去抑制效应”很好地回答了这一问题，舒勒以为有六个因素在相互作用，使得个人在网络上更放纵。

  1. 匿名。

    网络的匿名性使得个人得以将线上的自我与其它时候的自我分离开来，他们可以避免对网络上的行为负责，想说什么就说什么，从而形成一个被分割的自我，不用受到道德的限制。

  2. 不可见。

    即使交流双方的身份是明确的，幕后的交流也更让人放松，面对面和目光的接触总会让人感到拘谨，对方的神态、动作、语气等不满或者冷漠的迹象也会抑制人们愿意表达的东西。

  3. 非对称。

    网络的讨论是间断的，对方必须要等你完整表述以后才能给出回复，而且可能要很久才会回复，往往就使人得不到反馈来改变思路，从而更容易钻入牛角尖，此外，你也可以发泄自己的情绪然后完全不理会对方的反馈。

  4. 自我代入。

    当个人在阅读文字时，会在脑海里用自己的声音读出来，从而无意识地把双方的交流视为自言自语，就像许多人经常在幻想中上演的“小剧场”一样，在这里，你可以不用受到任何外部的约束。

  5. 想象力。

    有些人会把他们的网上生活看作一场游戏，与现实世界的要求和责任分开，匿名有助于这种想象，与匿名的影响不同的是，想象会创造出一个更加复杂、不一样的自我。

  6. 缺乏权威。

    面对权威人物，人们不愿意说出他们的真实想法，担心受到处罚，但是在网络上，你看不到他们的衣着、装饰、环境，现实的地位和权威被弱化了，在感觉上更为平等，因而人们更容易说出自己的想法。

三、一种亚文化

  尽管每个人都会被网络去抑制，但是总有部分缺乏同理心的人表现得更加突出，挑衅他人、扭曲讨论而且乐此不疲，这些不顾及他人感受而以骂战为乐的行为，久而久之就形成了一种亚文化。

  BBS时代就有引战高手发布《BBS论战指南》和《引战十二戒律》来引导想要涉足这一圈子的人，后者第十二条戒律写道：“犹豫不决时，开骂。如果你不记得前面十一条，记住这条就够了。”

  如今的互联网中又流传着“典、孝、急”三宝，用来在骂战中居于不败地位，使用方法就是说对方的长篇大论是重复烂话、某人的支持者是他的孝子、试图与你争论的人是急了。我们可以很清楚地看到有这么一群人在研究如何进行骂战。

  这样一种以惹怒他人为乐的亚文化有时会变得极其恶劣，比如诱导癫痫患者去看闪光图片来使他们发病、恶搞死者的形象、曝光他人的隐私等等。在外人看来难以容忍的恶行，对于这一群体而言，只是“玩玩而已”，是给受害者上了“人生中重要的一课”。

  这一亚文化群体固然讨厌，但是当我们在谈及他们的时候，也要当心自己有可能无意地成为他们中的一员，破坏正常的社区秩序。

四、每个人都有可能

  不论有意还是无意，网络上的讨论常常会变得野蛮，早在互联网还是阿帕网的时候，这种情况就出现了，使用者为此发明了第一个表情符号笑脸：- ）来表达友善，但却被接收者以为是嘲讽而加深了误会。

  实证研究表明，受外部环境的影响，每个人都可能变成一个“喷子”。当一个人处于负面情绪时，更有可能发表不当的言论，引战的高峰发生在一周的周一和每天的晚上，低谷则是在周六和上午，这与一个人的情绪变化是相关的。

  负面情绪是会传染的。个人发表的言论还会受到已有讨论的影响，引战的贴子会引来更多的引战言论，而且普通的用户在参与到一个骂战后，更有可能在接下来的讨论中有不当言论。

  所以，当我们在使用社交媒体时，除了避免与那些引战用户接触以外，还应该注意到自身的情绪影响，尽量不要使用讽刺性或者容易引起误会的表达，笑脸的表情并不能拯救一切。
